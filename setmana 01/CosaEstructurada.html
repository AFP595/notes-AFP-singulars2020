Estructura Básica de un Documento HTML

La estructura básica de un docuento HTML es la siguiente:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        body {
            margin: auto;
            max-width: 1024px;
        }
    </style>
</head>
<body>
    (Cuerpo de la página)
</body>
</html>

La etiqueta head contiene toda la configuración del archivo: título e icono en la pantalla del navegador, conjunto de caracteres, etc. La etiqueta body contiene el cuerpo del documento.

Los comentarios en un documento HTML se escriben dentro de <!-- comentario -->, y en un documento CSS se escriben dentro de /* comentario */.
